  * [ ] **3. 实现模块二界面层 (Widget)**:

      * 打开 `module2/module2_widget.py` 文件。
      * 创建一个名为 `Module2Widget` 的 `qt.QWidget` 类。

    <!-- end list -->

    ```python

    class Module2Widget(qt.QWidget):
    ```

#### **第二阶段：模块集成与导航**

  * [ ] **4. 实现模块二适配器 (Adapter)**:

      * 打开 `module2/module2_adapter.py` 文件。
      * 参考 `module1/module1_adapter.py`，创建一个 `Module2Adapter` 类，并继承 `ModuleInterface`。
      * 实现 `ModuleInterface` 要求的所有抽象方法：
          * `get_module_name()`: 返回 `"module2"`。
          * `get_display_name()`: 返回 `"引导式分割"` (或您希望在按钮上显示的任何文本)。
          * `create_widget()`: 导入并返回一个 `Module2Widget` 的实例。
          * `get_dependencies()`: 返回 `["module1"]`，明确其在工作流中的依赖关系。
      * *目标：将模块二的组件和逻辑“适配”到插件的模块管理系统中。*

  * [ ] **5. 注册模块二**:

      * 打开主插件文件 `tavi_analytics/tavi_analytics.py`。
      * 在 `_initialize_modules` 方法中，模仿注册模块一的代码，添加注册模块二的逻辑。

    <!-- end list -->

    ```python
    # tavi_analytics.py in tavi_analytics class

    def _initialize_modules(self):
        try:
            manager = ModuleManager()
            
            # ... (注册模块一的代码)

            # 注册模块二
            if self._config.is_module_enabled("module2"): # 假设未来会用配置控制
                from module2.module2_adapter import Module2Adapter # 导入适配器
                module2_info = ModuleInfo(
                    name="module2",
                    display_name="引导式分割", # 与适配器中保持一致
                    module_class=Module2Adapter,
                    dependencies=["module1"],
                    enabled=True
                )
                manager.register_module(module2_info)
                logging.info("模块二注册成功")
        
        except Exception as e:
            # ...
    ```

      * *目标：让主程序知道模块二的存在，从而在UI上创建它的导航按钮。*
